Users can test their typing speed, accuracy, and track their personal best score.

## 🎯 Features

* ✅ Start typing test by button or typing directly
* ✅ Difficulty levels: Easy / Medium / Hard
* ✅ Modes:

  * Timed (60 seconds)
  * Passage mode (no time limit)
* ✅ Real-time statistics:

  * Words Per Minute (WPM)
  * Accuracy %
  * Time
* ✅ Visual feedback:

  * Correct letters → Green
  * Wrong letters → Red + underline
* ✅ Result screen after completion
* ✅ Personal best saved using **localStorage**
* ✅ Responsive design for mobile & desktop
* ✅ Uses provided Frontend Mentor assets
